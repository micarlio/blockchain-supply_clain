# Blockchain Distribuida para Rastreabilidade

Projeto da disciplina voltado para a construcao de uma blockchain distribuida com `Proof of Work`, usando como dominio a rastreabilidade de produtos em uma cadeia de supply chain.

A ideia geral do sistema e registrar eventos imutaveis relacionados ao fluxo de produtos, lotes e transformacoes ao longo da cadeia.
